# Total Fitness - Twilio Voice Server

This server triggers support calls from the Total Fitness Kiosk to staff using Twilio Programmable Voice.

## 📦 Requirements
- Node.js
- Twilio account

## 🛠️ Setup

1. Clone the repo
2. Create a `.env` file based on `.env.example`
3. Run the app:

```bash
npm install
node server.js
